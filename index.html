PROJECTS

SafeDrive

This app helps travelers and tourists in new areas and locations drive safely through hills containing sharp turns and steep inclines, even in low visibility and with the assistance of timely alerts. It includes real-time location tracking, ability to add favorites, minimalistic UI for a distraction-free driving experience, and toggleable settings including dark mode. Simply set a destination and you're ready to roll!

For the steep slopes, Elevation API offered by Google Cloud Console has been used. For the sharp turns, Polyline API was used to generate a route to the destination after which intensive logic was written to identify the various types of sharp turns in the route. The ability to add and remove favorites has been made possible using Firebase, which also provides the authentication service uesd in the app.

Tech stack

  • Flutter / Dart
  • Google Maps API
  • Firebase
